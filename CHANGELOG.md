# Changelog

This file lists user-visible changes by release.

## 1.0.0 - 2026-05-05

Initial release.

Added:

- Desktop app for supported link preparation.
- Queue with pending, active, ready, and failed states.
- Configurable concurrency.
- Configurable retry count.
- Headless browser mode.
- Ready-link copy action.
- Ready-link export action.
- Python Playwright engine.
- macOS, Windows, and Linux build scripts.
